# Tech Book Club Landing Page - Frontend Mentor

This is a solution to the Tech Book Club landing page challenge. It is a fully responsive layout that showcases membership options, community benefits, and a step-by-step joining process.

## Features

- **Responsive Flexbox Layout:** Used CSS Flexbox to create a flexible structure that adapts from a single-column mobile view to a complex desktop arrangement.
- **Custom Visual Effects:** Implemented a unique gradient text overlay on the main headline (`background-clip: text`) to match the design aesthetics.
- **Adaptive Media Queries:** Optimized the layout for three distinct screen sizes: Mobile (< 670px), Tablet (670px - 1170px), and Desktop.
- **Semantic HTML:** Built using accessible tags like `<main>`, `<section>`, and `<footer>` to ensure better SEO and structure.

## Technologies Used

- HTML5
- CSS3 (Flexbox, CSS Variables, & Media Queries)
- Git & GitHub
